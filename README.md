<h2>Project Overview</h2>

<p>
The <b>Student Performance Analyzer</b> is a Python-based console application designed to manage student records, analyze academic performance, generate statistical reports, and create visual charts.
</p>

<p>
The project demonstrates the practical implementation of:
</p>

<ul>
  <li>Object-Oriented Programming (OOP)</li>
  <li>File Handling using CSV</li>
  <li>Data Analysis with Pandas</li>
  <li>Data Visualization with Matplotlib</li>
  <li>Python Functions & Modular Programming</li>
</ul>

<hr>

<h2>Features</h2>

<ul>
  <li>Add Student Records</li>
  <li>Remove Students</li>
  <li>Search Students by ID or Name</li>
  <li>Update Student Information</li>
  <li>Generate Performance Reports</li>
  <li>Create Professional Charts & Visualizations</li>
  <li>Save Data Automatically to CSV File</li>
  <li>Automatic Grade Calculation</li>
  <li>Statistical Analysis using Pandas</li>
</ul>

<h2>⚙️ Installation</h2>

<h3>1️⃣ Clone the Repository</h3>

```bash
git clone https://github.com/mharoon1434/Student-Performance-Analyzer-Py.git
```

<h3>2️⃣ Open Project Folder</h3>

```bash
cd student-performance-analyzer
```

<h3>3️⃣ Install Required Libraries</h3>

```bash
pip install pandas matplotlib numpy
```

<hr>

<h2>▶️ Run the Project</h2>

```bash

<h2>🧮 Grade Calculation Formula</h2>

<p>
The final score is calculated using the following formula:
</p>

```text
Final Score = (0.4 × Midterm) + (0.5 × Final Exam) + (0.1 × Assignment)
```
